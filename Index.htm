            <h2 class="text-4xl font-bold mb-8 text-blue-700">¿Qué es un indicador?</h2>

            <ul class="list-disc list-inside text-left text-lg space-y-3">

                <li>Expresión cuantitativa, observable y verificable. </li>

                <li>Permite describir características de un servicio, comportamientos o fenómenos de un proceso. </li>

                <li>Se logran con la medición de una variable o relación entre variables que se registra, procesa y presenta como información relevante para saber si eso que midió avanza, se mantiene estable o retrocede a lo largo del tiempo. </li>

                <li>También permite compararse con una meta, un ideal (benchmark) o un requerimiento y hasta con uno mismo, para saber si la práctica o prestación es de calidad. </li>

                <li>Si el indicador se desvía del objetivo se deben tomar acciones de mejora o acciones preventivas según el caso. </li>

                <li>Apoyan la toma de decisiones y las evaluaciones. </li>

            </ul>

        </div>

        <div class="slide-content">

            <h2 class="text-4xl font-bold mb-8 text-blue-700">Proceso para la construcción de un buen indicador</h2>

            <ul class="list-disc list-inside text-left text-lg space-y-3">

                <li><strong>Monitores:</strong> quién mide y quién valida.  Sector responsable del indicador. </li>

                <li><strong>Indicador:</strong> título de la medida. Es importante que el nombre del indicador sea claro, conciso, corto y auto explicativo. </li>

                <li><strong>Razón para medirlo:</strong> justificación de la importancia del indicador. </li>

                <li><strong>Numerador:</strong> describa el numerador para el cálculo del indicador si aplica (por ejemplo en porcentaje). </li>

                <li><strong>Denominador:</strong> describa el denominador para el cálculo del indicador si aplica. </li>

                <li><strong>Unidad de medida:</strong> corresponde al parámetro o unidad de referencia para determinar la magnitud de medición del indicador (razón, porcentaje, casos, actividades, entre otros). </li>

                <li><strong>Tipo de indicador:</strong> si es de estructura, proceso, resultado, exigido por JCI u otra normativa (ministerio, bomberos, etc.). </li>

                <li><strong>Dimensiones que evalúa:</strong> eje o línea estratégica. </li>

                <li><strong>Población:</strong> población a evaluar, si es total o muestra y si es muestra cálculo de la muestra. </li>

            </ul>

        </div>

    </div>

            <h2 class="text-4xl font-bold mb-8 text-blue-700">Proceso para la construcción de un buen indicador</h2>

            <ul class="list-disc list-inside text-left text-lg space-y-3">

                <li><strong>Método de recolección de los datos:</strong> manual, por observación, análisis de registros o electrónico por el sistema. </li>

                <li><strong>Frecuencia de recolección:</strong> señale la frecuencia con la cual se recolecta la información de avances y a partir de la cual se realiza el reporte de avance. </li>

                <li><strong>Persona responsable:</strong> persona o personas que tendrán a su cargo la gerencia del indicador (que deben velar por que se cumplan las metas definidas), así como la elaboración del mismo y el reporte periódico de la información de avance de este indicador en los términos presentados en la ficha técnica.  Debe contener los datos de contacto de los responsables. </li>

                <li><strong>Meta o estándar de cumplimiento:</strong> cuál es el objetivo que se pretende alcanzar para la medida que se está reportando. </li>

                <li><strong>El indicador incluye y excluye:</strong> aquellos pacientes, atributos, situaciones, unidades, etc. que están contempladas en la medición y aquellas que quedan excluidas. </li>

                <li><strong>Observaciones:</strong> Comentarios que deban tenerse en cuenta sobre el indicador, que se consideran pertinentes y que no fueron recogidos a través de las otras secciones de la ficha técnica.  En tal sentido, aquí se incluyen comentarios que se consideren pertinentes para la conceptualización y comprensión del indicador. </li>

                <li><strong>Validación:</strong> Proceso de verificación de la exactitud y fiabilidad de los datos. </li>

            </ul>

        </div>

            slides.forEach((slide, i) => {

                slide.classList.remove('active');

            prevBtn.disabled = currentSlide === 0;

            nextBtn.disabled = currentSlide === slides.length - 1;

        }

 

        prevBtn.addEventListener('click', () => {

            if (currentSlide > 0) {

                currentSlide--;

                showSlide(currentSlide);

            }

        });

 

        nextBtn.addEventListener('click', () => {

            if (currentSlide < slides.length - 1) {

                showSlide(currentSlide);

            }
